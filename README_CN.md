## 什么是 Fine

&emsp;&emsp;Fine 是帮助你快速生成静态网站的工具，你可以用它为自己的项目快速生成文档而不用关心样式、部署的问题。

&emsp;&emsp;甚至你可以用 fine 来生成自己的博客，fine 为任何一个项目都提供了独特的域名。

&emsp;&emsp;Fine 也拥有全球专业级别 CDN。当然，最重要的是，fine 是免费的。

<br/>
